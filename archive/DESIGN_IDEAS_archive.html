Progression XP & Leveling

XP Curve
10 * level ^ 1.3
Exponential ramp. Lv1: 10 XP, Lv5: ~85 XP, Lv10: ~200 XP, Lv20: ~500 XP
  • XP gems drop from every enemy kill
  • Gem value = enemy XP value (scales with time)
  • Gems attracted via pickup area (boosted by Attractorb)
  • God mode: each pickup = 50% of next level
Level-Up Choices
  • Each level: choose 1 of 3 random options
  • Options: new weapon, weapon upgrade, new passive, passive upgrade
  • Cards color-coded by type (silver/gold/green/magenta)
  • 3 rerolls per run (infinite in god mode)
  • 6 weapon slots + 6 passive slots max
  • Multi-level-ups queued via pending_levels

Architecture 5 Autoloads

SignalBus
10 signals. Decoupled event bus for all game systems.
GameManager
HP, XP, leveling, inventories, stat multipliers, evolution checks, god mode.
PoolManager
9 object pools. Acquire/release pattern. Double-release guards.
AudioManager
16 pooled AudioStreamPlayers, 50ms dedup, wire_button_sfx() helper.
UiTheme
Kenney fantasy-ui-borders textures, Aldo the Apache font, ornate panels/buttons/slots.
Key Patterns
  • distance_squared_to for nearest-enemy (no sqrt)
  • UI panels: PROCESS_MODE_ALWAYS (work while paused)
  • Enemy spawner: manual delta accumulator, not Timer nodes
  • Tweens for flash effects (not await)
  • Pooled entities: process_mode=DISABLED, visible=false when released
  • Entities removed from groups on release, re-added on acquire
  • _is_dead flag prevents double-death from concurrent hits
  • on_hit(hit_enemy) pattern avoids splash self-damage

Settings & UI Flow Implemented

Settings Panel
  • Master volume slider (0–100%)
  • SFX volume slider (0–100%)
  • Resolution dropdown (540p / 720p / 1080p / 1440p)
  • Fullscreen toggle (checkbox)
  • God Mode toggle (checkbox, not persisted)
  • Persists to user://settings.cfg via ConfigFile
Screen Flow
Main Menu
  ├ Start → Game
  ├ Settings → Settings Panel
  └ Quit

Game (WASD/Arrows move, auto-fire)
  ├ ESC → Pause Menu
  │  ├ Resume
  │  ├ Settings
  │  └ Main Menu
  ├ Level Up → 3-Card Select
  └ HP=0 → Game Over
      ├ Retry
      └ Main Menu
God Mode Details
  • Player takes zero damage
  • XP gain = 50% of next level per pickup
  • Infinite rerolls during level-up
  • Heal to full when toggled on mid-game
  • Not persisted to settings file (resets each launch)
